Verkefni 1 - Að búa til vefsíðu með HTML

Veldu eða smíðaðu html sniðmát fyrir vefsíðuna þína, skipulegðu hana með upplýsingum um þína ferilskrá. Skrásettu fyrsta verkefnið og settu á github með því að nota Git bash.

Hönnunarforsendur

  • Síðan sé einföld til notkunnar og allar upplýsingar séu á áberandi staðsettar.
  • Síðan sé ekki of lík sniðmáta sem ég nota sem grunn fyrir síðuna og gera hana meira að minni síðu.
  • Ferilskráin mín sé sett fram á skýran máta svo ég geti nýtt mér hana í framtíðinni.

Hugbúnaður

Við gerð þessarar síðu er notast við:

  • Visual Studio Code hér
  • GitHub Desktop hér
  • GitHub.com hér

Val á sniðmáti

Til að byrja verkefnið fór ég á síðuna HTML5 UP , þar er fullt af sniðmátum sem má nota frítt. Þegar ég var búin að skoða sniðmátin valdi ég eitt þeirra til að byrja með en þegar ég sá að valglugginn gat ekki innihaldið 6 verkefni án þess að að gjörsamlega breyta sniðmátinu. Ég reyndi að breyta því en þá enduðu valmöguleikarnir fyrir aftan nafnið mitt og þar með ekki hægt að fara á þær síður þar sem nafnið er linkur að forsíðunni.

Því ákvað ég að skoða önnur sniðmát með það í huga hvað ég þarf margar tengingar efst á síðunni. Þá fann ég sniðmátið Landed sem virðist henta vel og er eiginlega flottara en það gamla.

Breyta HTML kóðanum

Áður en ég setti síðuna á GitHub setti ég hana upp eins og ég vill hafa hana. Til að breyta síðunni notaði ég Visual Studio Code þar sem ég nota það nú þegar til að Tekka og kóða í Python. Til að geta skoðað síðuna jafnóðum við breytingar notaði ég pakka frá Microsoft sem heitir Live Preview . Sá pakki gerir mér kleift að skipta skjánum upp í tvennt þar sem ég get kóðað öðru megin og séð breytinguna sjálfkrafa uppfærast á hinum helmingnum.

Það sem er einnig hentugt við þetta sniðmát er að þaðer fullt af mismunandi sýnidæmum um hvernig hægt er að setja fram hluti sem ég nýtti mér t.d. gera bullet point, hvernig skil milli hluta á síðuni eiga að vera, uppsetning texta o.s.fl.

Forsíða

Hér setti ég til að byrja með mínar eigin myndir til að hafa í bakgrunn og myndina af mér fremmst á síðunni. Eftir það set ég inn minn eigin text með viðeigandi tilvitnunum. Neðst á síðunni var fullt af auka dóti sem ég tók út þar sem ég sá enga þörf í því. Efst á síðunni breytti ég borðanum svo hann hafi: Heim, Ferilskrá og lista sem fellur með öllum verkefnunum.

Ferilskrá

Til að byrja með skrifaði ég ferilskránna mína beint inn en fannst það ekki líta nógu vel út svo ég fór að Googla sniðmát fyrir ferislkrá í HTML og datt inn á þetta frá Walter Patterson. Fyrst reyndi ég að nýta afrita smá part úr hans sniðmáti en það tókst ekki svo ég endaði á að sameina CSS, Java og HTML kóðana sem svín virkaði. Þá fór ég að breyta textanum og aðlaga hann að ferilskránni minni og eyða því út sem mér fannst óþarfi.

GitHub

Þá er komið að því að setja síðuna á GitHub. Fyrst bjó ég mér til aðgang og gerði (e.repository) fyrir alla kóðana sem eru tengdir síðunni. Í gegnum það get ég síðan byrt síðuna. En fyrst þá þarf ég að koma kóðanum úr tölvunni yfir á GitHub. Til þess notaði ég GitHub Desktop sem býður uppá viðmót milli Github og möppu í tölvunni. Í gegnum það get ég sent og sótt skjöl á milli með þægilegu viðmóti.

Eins og sést á myndinni þá er takki inni í forritinu sem gerir mér kleipt að opna möppuna af GitHub í Visual Studio Code og breytt síðunni þar. Eftir að hafa breytt eitthverju á síðunni er valið (Commit to main) sem er í neðra vinstra horninu með lýsingu á hverju var breytt.

Setja upp vefsíðu

Inná GitHub fer ég undir Settings í Pages þar sem valið er Brach main og vistað eftir það þá ætti síðan að vera komin á netið undir léni sem stendu á með grænum bakgrunn.
